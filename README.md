# Roda Mailer Extensions

A few helpful extensions to the Roda mailer plugin.

## Installation

Add this line ot your application's Gemfile:

```ruby
gem "roda-mailer_ext"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roda-mailer_ext
